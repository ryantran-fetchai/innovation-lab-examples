# Receipt / Expense Calculator

Split receipts fairly with a photo or manual list. Send a receipt image and I extract line items; then everyone replies with which items they brought and I show the fair split (only people who brought an item share its cost).

## Features

- **Receipt photo:** Attach an image → I list items and prices.
- **Manual items:** Add items with `add &lt;name&gt; &lt;price&gt;`.
- **Poll:** Each person replies with item numbers (e.g. 1,2,3).
- **Fair split:** Say `calculate` to see each person’s share.

## Usage

1. Send a **photo of your receipt** or say **new receipt** and **add Pizza 12**, etc.
2. Say **done** to start the poll.
3. Each person replies with the **numbers** of items they brought (e.g. 1,2,3).
4. Say **calculate** to see the split.

## Commands

- **help** — Show full instructions  
- **new receipt** — Start over  
- **add &lt;name&gt; &lt;price&gt;** — Add an item  
- **done** — Start the poll  
- **1,2,3** — Your items (during poll)  
- **calculate** — Show split
